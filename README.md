# TetrisMD
A Homebrew Tetris Game for the Sega Megadrive/Genesis, write with C and using SGDK. 
（自制SEGA MD平台的俄罗斯方块游戏，用C和SGDK引擎开发。） 
## 操作方法：
|    按键    |           操作           |
|:----------:|:------------------------:|
| 方向键左右 | 控制方块左右             |
| 方向键上   | Hard Drop                |
| 方向键下   | Soft Drop                |
| A          | Hold(保留或替换方块)     |
| B          | Rotate Left(方块左旋转)  |
| C          | Rotate Right(方块右旋转) |
## 运行方法：
用Sega Megadrive/Genesis模拟器（Fusion、PicoDrive、Gens...）加载rom运行。
## 编译：
SGDK 1.41
